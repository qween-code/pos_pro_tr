## ğŸš€ Quick Start

### Prerequisites

- **Flutter** 3.16+ ([Install](https://flutter.dev/docs/get-started/install))
- **Docker** & Docker Compose ([Install](https://docs.docker.com/get-docker/))
- **Python** 3.11+ (for backend development)
- **PostgreSQL** 15+ (or use Docker)

### 1. Clone Repository

```bash
git clone https://github.com/qween-code/pos_pro_tr.git
cd pos_pro_tr
```

### 2. Start Backend (FastAPI + PostgreSQL)

```bash
cd backend
docker-compose up -d
```

This starts:
- PostgreSQL (port 5432)
- Redis (port 6379)  
- FastAPI (port 8000)
- pgAdmin (port 5050) - Optional
- Redis Commander (port 8081) - Optional

**Access Swagger API Docs:** `http://localhost:8000/docs`

### 3. Run Flutter App

```bash
cd ..  # Back to project root
flutter pub get
flutter run
```

**Default Credentials:**
- **Admin:** `admin@pospro.com` / `admin123`
- **Cashier:** `cashier@pospro.com` / `cashier123`
